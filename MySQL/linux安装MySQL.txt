linux下如何安装多个mysql数据库
 此处以mysql5.1.63为例：
1.下载相应的mysql软件包。
1）系统平台：
uname -a  -> 查看系统版本
cat /proc/cpuinfo | grep model -> 查看处理器
2）下载：
http://dev.mysql.com/downloads/ -> MySQL Community Server
或者：
http://downloads.mysql.com/archives/community/
可以选择source code，即*.tar.gz压缩包
2.存放，解压和链接
确定存放目录，常用的有：/user/local, /home/, etc. 根据需要和习惯
例如：/home/dapeng/mysql
tar -zxvf mysql-5.1.63.tar.gz 
ln  -s /home/dapeng/mysql-5.1.63 /home/dapeng/mysql
(可省略) rm -rf mysql-5.1.63.tar.gz
3.配置文件
cd /mysql-5.1.63
cp support-files/my-medium.cnf /home/dapeng/mysql/my.cnf
vi 修改my.cnf,data库要足够大(数据库文件默认在mysq/var/下)
# The following options will be passed to all MySQL clients
[client]
#password       = your_password
port            = 6363
socket          = /home/dapeng/mysql/tmp/mysql.sock

# Here follows entries for some specific programs

# The MySQL server
[mysqld]
port            = 6363
socket          = /home/dapeng/mysql/tmp/mysql.sock
#socket         = /tmp/mysql.sock
4.安装mysql数据库
在/home/dapeng/mysql目录下：
./bin/mysql_install_db --default-file=my.cnf
注意：会出现用户设置提示，请跳过
5.启动mysql server(mysql daemon启动)
./bin/mysqld_safe --defaults-file=my.cnf &
6.创建root账户
./bin/mysqladmin --defaults-file=my.cnf -u root password "123456"
7.创建新的数据库和新用户
（1）以root登陆mysql
./bin/mysql  --defaults-file=my.cnf -u root -p
（2）创建数据可和新用户，如：
mysql>create database demo;
mysql>grant select,insert,update,delete,create view,create,index,drop on demo.* to test@localhost;
mysql>set password for test@localhost=password('123456');
8.以test登陆mysql
./bin/mysql --defaults-file=my.cnf -u test -p 进入
mysql>
9.至此结束。
在/home/dapeng/mysql目录下：
./bin/mysql_install_db  --defaults-file=my.cnf --basedir=/home/dapeng/mysql --datadir=/home/dapeng/mysql/data (--user=mysql或者尝试其他用户省略) 
启动和关闭数据库：
./bin/mysqld_safe --defaults-file=my.cnf & //启动
./bin/mysqladmin -u root -p shutdown 	     //关闭
